# dayaheadcolloector
Simple script to fetch dayahead prices.

This is still under construction. No stable API, only supoort Sweden SE3.

## `.env`-file
The database settings are read from the `.env`-file, which is not included.

Sample `.env`-file:
```
DB_HOST=192.168.1.27
DB_USER=el
DB_DB=el
DB_PASSWORD=password

```
